La Configuration globale de Git

À faire une fois pour toutes

De l'installation à GitHub CLI en 10 étapes simples

Jack-Kevin TSAKEM
VinTech

Ce guide vous accompagne pas à pas dans la configuration complète de Git sur votre machine. Que vous soyez sur macOS, Linux ou Windows, vous trouverez toutes les commandes nécessaires pour devenir opérationnel en quelques minutes.

10
Étapes
3
Plateformes
15min
Temps requis
1
Vérifier l'installation de Git
git --version
Vérifier si Git est déjà installé. Sans Git, rien ne fonctionnera.
💡 Si Git n'est pas installé, passez à l'étape suivante.
Jack-Kevin TSAKEM
VinTech
2
Installer Git
🍎 macOS
brew install git
🐧 Ubuntu/Linux
sudo apt update && sudo apt install git
🪟 Windows
Télécharger depuis git-scm.com
Installer Git sur votre système d'exploitation s'il est absent.
JK
Jack-Kevin TSAKEM
VinTech
3
Configurer votre identité
git config --global user.name "Jack-Kevin TSAKEM" git config --global user.email "jack.kevin@vintech.com"
Configurer votre identité. C'est obligatoire pour signer vos commits.
💡 L'email doit correspondre à votre compte GitHub.
JK
Jack-Kevin TSAKEM
VinTech
4
Générer une clé SSH
ssh-keygen -t ed25519 -C "jack.kevin@vintech.com"
Générer une nouvelle clé SSH pour une authentification sécurisée sans avoir à taper votre mot de passe à chaque fois.
💡 Appuyez sur Entrée pour accepter l'emplacement par défaut et définir un mot de passe (optionnel).
JK
Jack-Kevin TSAKEM
VinTech
5
Démarrer l'agent SSH
eval "$(ssh-agent -s)" ssh-add ~/.ssh/id_ed25519
Démarrer l'agent SSH et charger votre clé privée pour la session en cours.
JK
Jack-Kevin TSAKEM
VinTech
6
Copier la clé publique SSH
🍎 macOS
pbcopy < ~/.ssh/id_ed25519.pub
🐧 Linux
xclip -sel clip < ~/.ssh/id_ed25519.pub
🪟 Windows (Git Bash)
cat ~/.ssh/id_ed25519.pub | clip
Copier votre clé publique SSH dans le presse-papiers.
💡 Sur Linux, vous devrez peut-être installer xclip : sudo apt install xclip
JK
Jack-Kevin TSAKEM
VinTech
7
Ajouter la clé SSH à GitHub
1. Aller sur GitHub.com 2. Settings → SSH and GPG keys 3. New SSH key 4. Coller votre clé 5. Add SSH key
Ajouter votre clé publique à votre compte GitHub pour qu'il puisse authentifier votre machine.
JK
Jack-Kevin TSAKEM
VinTech
8
Tester la connexion SSH
ssh -T git@github.com
Tester la connexion SSH avec GitHub pour vous assurer que tout est correctement configuré.
💡 Vous devriez voir un message de bienvenue de GitHub.
JK
Jack-Kevin TSAKEM
VinTech
9
Installer GitHub CLI
🍎 macOS
brew install gh
🐧 Ubuntu/Linux
sudo apt install gh
🪟 Windows
winget install --id Github.cli
Installer le GitHub CLI pour gérer vos projets GitHub (pull requests, etc.) directement depuis le terminal.
JK
Jack-Kevin TSAKEM
VinTech
10
Authentifier GitHub CLI
gh auth login
Authentifier le GitHub CLI pour lui donner accès à votre compte GitHub.
💡 Suivez les instructions à l'écran pour terminer l'authentification.
JK
Jack-Kevin TSAKEM
VinTech
